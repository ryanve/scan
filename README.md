# [scan](../../)

querySelectorAll query engine

## API

#### `scan(selector|nodes, context?)` &rarr; array

```js
scan('.example') // Select elements.
scan(nodeList) // Arrayify nodeList.
```

#### `.find()`

- `scan.fn.find.call(collection, selector)`
